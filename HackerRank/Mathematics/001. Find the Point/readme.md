# Title: Find the Point
## Official Level: Easy

# Content:
Consider two points, p=(p_x, p_y) and q=(q_x,q_y).
We consider the inversion or point reflection, r=(r_x, r_y), of point p across point q to be a 180 degree rotation of point p around q.

## Sample Input
```
2
0 0 1 1
1 1 2 2
```

## Sample Output
```
2 2
3 3
```
